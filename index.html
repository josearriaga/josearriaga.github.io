Projects

This Raspberry Pi cluster, which I named "Anton" after the fictional character Bertram Gilfoyle's server name, is a Beowulf-style cluster which has a main purpose of distributing work loads to any of the 16 mini computers. This project was the basis for our Creative Computer Project in the SkillsUSA competition. The underlying code base uses the mpi4py. As us computer scientists know, the message passing standard is, as the name states, a method of passing a message from point A to point B. To put it simply, MPI is a high level network programming abstraction. mpi4py takes this and and provides bindings of the message passing standard for the Python Programming language. More info on this topic can be found here and here.

My current project is a scientific payload for a 1 pound 1 mile high rocket competition with the Tripoli Rocketry association.
